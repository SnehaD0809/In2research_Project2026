# In2research_Project2026

# Main Project Focus
Building an Agent-Based Model (ABM), which is a giant virtual simulation of how blood gets from a donor, to a processing center, and finally to a hospital trust. They want to simulate what happens to that blood supply chain when a massive flood or heatwave hits.

# My Objectives

1.Predict Cost - Use NHS ERIC data + climate exposure data (flooding/overheating) to find patterns in what drives costs, and compare modelling approaches (Bayesian vs ML). What factors drive NHS estate costs, and how much climate shocks influence them? Climate exposure increases financial burden.

2.Predict Climate Incidents - What makes an NHS site more likely to experience flooding or overheating risk? Which site characteristics determine climate vulnerability
